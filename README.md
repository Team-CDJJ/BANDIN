## 4. 기타 프로젝트 관련 사항

### 4.1 커밋 컨벤션

```
- feat: 새로운 기능 구현
- fix: 오류 수정
- docs: readme.md, json 파일 등 수정, 라이브러리 설치 (문서 관련)
- refactor: 코드 리팩토링
- style: 코드에 변화가 없는 수정 (세미콜론 등)
- chore: 빌드 부분 혹은 패키지 매니저 수정 사항
- rename: 파일 혹은 폴더명 수정, 옮기기
- remove: 파일 삭제
```

### 4.2 코드 컨벤션

```
- 컴포넌트 함수 선언 시 화살표 함수 사용
- 파스칼 케이스
```
